# 寶可夢外掛管理器
這個是我的第一個Android APP，如有問題歡迎詢問
# 關於這個APP
1. 整合寶可夢外掛的下載與管理功能，方便用戶直接從管理器更新或安裝。
2. 加入檢測Magsik與檢測是否使用虛擬定位(測試版本使用新版檢測方式)
3. 使用material design 3 並使用動態顏色服務(Android 12以上支援)

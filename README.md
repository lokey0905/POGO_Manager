# 寶可夢外掛管理器
這個是我的第一個Android APP，使用koltin編寫，先前都還沒用過java與koltin，編寫方式大部分都是Google與使用github copilot
# 關於這個APP
- [x] 整合寶可夢外掛的下載與管理功能，方便用戶直接從管理器更新或安裝。
- [x] 加入檢測Magsik與檢測是否使用虛擬定位(測試版本使用新版檢測方式)
- [x] 使用material design 3 並使用動態顏色服務(Android 12以上支援)

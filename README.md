# Pogo管理器
這個是我的第一個打code製作的Android APP，編寫方式大部分都是Google搜尋與使用AI輔助

# 關於這個APP
- [x] 整合Pokémon寶可夢相關軟體下載與檢測更新功能
- [x] 整合Monster Hunter Now魔物獵人相關軟體下載與檢測更新功能
- [x] 加入檢測Magisk路徑與檢測裝置定位等
- [x] 使用Material Design 3 
- [x] 可供下載修改部分中文化版本APP

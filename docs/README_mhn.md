## 各種授權列表
每一天，一張授權，Basic或Premium，只能支援，同一個時間，一台手機(一個Device)，每一天，可以轉MHN帳號三次，每次按「切換」/「更換」，以GMT 0，開始計算。

GMT 0，即是北京/台灣時間，上午8時。
